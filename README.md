# AI 搜索产品经理 Skill V2.0
## 简介
专为 AI 搜索产品经理打造的通用专业技能包，覆盖需求拆解、搜索质量评估、Bad Case分析、策略优化、PRD方案输出、跨团队协作全工作场景。

## 适配平台
扣子 Coze、Cursor、GPT、各类国内大模型、Claude
**纯文本通用指令，无网络限制、无平台绑定、可自由迁移**

## 核心能力
1. 用户搜索意图深度拆解
2. 搜索结果四维量化质量打分
3. Bad Case 根因定位与分级优化方案
4. 标准化产品策略文档输出
5. 算法&研发对接沟通规范

## 使用方法
1. 复制仓库内 `SKILL.md` 全部内容
2. 粘贴至任意大模型的【自定义指令 / 系统人设】
3. 直接对话即可启用全套专业能力
